\problemname{The Chocolate Box}
Bosse likes chocolate.
He therefore always has an opened chocolate box in the pantry.
When it runs out, he secretly buys a new one and pretends nothing happened.
Bosse's wife, who is suspicious by nature, is puzzled that the box never seems to run out.
She therefore begins to occasionally count the number of chocolate pieces remaining.
Write a program that, given her observations, computes the minimum number of new boxes Bosse must have bought during the period.

\section*{Input}
The first line contains an integer $N \le 100$, the number of observations.
Then follows a line with $N$ integers, the number of chocolate pieces in the box (between $1$ and $100$) at each observation, in the order they are made.

\section*{Output}
The program should output a single line with one integer: the minimum number of new boxes Bosse must have provably bought during the period.
